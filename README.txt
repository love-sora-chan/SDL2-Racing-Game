第31組專題-藤原填海
Initial F : Fill The Sea

Demo 影片連結:https://youtu.be/Y83hrpIebBo?si=JoMiSVIwGRfsV_7V

作者:
B12901067 黃子懿
B12901072 連善珩

操作：
↑方向鍵：操作按鈕/加速
↓方向鍵：操作按鈕/減速
←方向鍵：操作按鈕/往左開
→方向鍵：操作按鈕/往右開
Enter：下一頁/選擇
Esc：上一頁

主頁面：
START：進入選地圖的頁面
RECORD：紀錄頁面
SETTINGS：設定頁面

START：
選地圖、車子、難易度

RECORD：
查看紀錄

SETTINGS：
調整音樂、音效的音量
手動重置紀錄
退出遊戲

遊戲中：
撞上石頭或開出賽道都會結束遊戲

遊戲結束：
(如果沒有自動轉換請按任意鍵)

輸入名字：
允許輸入：字母或字母上方的數字(右側數字鍵盤無法輸入)
